# Gerenciamento de Tarefas TechFlow Solutions

## 📌 Objetivo do Projeto

Desenvolver um sistema de gerenciamento de tarefas que permita acompanhar o fluxo de trabalho em tempo real, priorizar atividades e melhorar a organização das equipes da TechFlow Solutions.

O sistema será baseado em metodologias ágeis, visando otimizar a produtividade e a gestão das atividades.

## 📊 Escopo

O sistema terá como funcionalidades principais:

* Criação de tarefas
* Visualização de tarefas
* Atualização de tarefas
* Remoção de tarefas

Essas funcionalidades seguem o modelo **CRUD (Create, Read, Update, Delete)** para gerenciamento das tarefas.

## 🛠️ Metodologia Adotada

Foi utilizada a metodologia ágil **Kanban**, com gerenciamento das atividades por meio do GitHub Projects.

As tarefas serão organizadas nas seguintes colunas:

* To Do (A Fazer)
* In Progress (Em Progresso)
* Done (Concluído)
